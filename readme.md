# Spelmotor för 2D-spel

Dokumentationen för denna spelmotor finns i mappen [docs](./docs). Den innehåller detaljerade beskrivningar av hur spelmotorn fungerar, dess arkitektur och exempel på hur man använder den för att skapa egna spel.

Alla filnamn är kopplad till en branch i Git-repot, så du kan enkelt navigera mellan olika delar av dokumentationen genom att byta branch.
För att byta branch kan du använda Git-kommandot:

```bash
git checkout <branch-namn>
```
